# Image Gallery with Lightbox Functionality

## Project Overview

This project is a responsive Image Gallery website developed using HTML, CSS, and JavaScript. The gallery allows users to browse multiple images in an organized layout and view them in full-screen mode using a Lightbox feature.

The project focuses on image presentation, responsive design, interactive user experience, and JavaScript-based image navigation.

---

## Features

* Responsive image gallery layout
* Interactive Lightbox functionality
* Full-screen image preview
* Next and Previous image navigation
* Keyboard navigation support
* Close button for exiting Lightbox
* Smooth image transitions
* Mobile-friendly design
* Attractive modern user interface
* Hover effects and animations

## Project Functionality

### Image Gallery

Displays multiple images in a responsive grid layout that automatically adjusts to different screen sizes.

### Lightbox View

Clicking on any image opens it in a full-screen Lightbox for better viewing.

### Next & Previous Navigation

Users can navigate through images using on-screen navigation arrows.

### Keyboard Navigation

Users can use keyboard arrow keys to move between images.

### Close Lightbox

A close button allows users to exit full-screen mode and return to the gallery.

### Responsive Layout

The gallery works smoothly on desktops, tablets, and mobile devices.
